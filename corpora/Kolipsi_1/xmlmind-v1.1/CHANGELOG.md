# Changelog

All notable changes to this project will be documented in this file.

## [v1.1](https://gitlab.inf.unibz.it/commul/Kolipsi/data/kolipsi-1/bundle/compare/v1.0...v1.1) - 2023-02-28

### Added

- 75 new transcripts (missing in first transcription), 135 new L2 texts
  (125 IT, 10 DE) from new transcripts

### Changed

- correct allcaps in texts
- minor changes in metadata format

## [v1.0](https://gitlab.inf.unibz.it/commul/Kolipsi/data/kolipsi-1/bundle/tree/v1.0) - 2021-05-05

### Added

- docs: Kolipsi 1 documentation, including transcription and annotation
  guidelines, proficiency levels descriptors and task instructions
- metadata: metadata for corpora, tasks, authors and texts as csv and
  xlsx
- xmlmind: corpus in custom xml scheme with inline annotations for
  errors, insertions, deletions, etc. (see docs for format description)
- mmax2: corpus with stand-off annotations produced in MMAX2
- annis: corpus for input in corpus query software ANNIS
- txt: original and corrected plain text versions of corpus
- pdf: original scans of handwritten texts (internal)
